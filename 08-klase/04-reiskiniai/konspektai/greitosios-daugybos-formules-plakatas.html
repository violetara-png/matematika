Greitosios daugybos formulės

Trys pagrindinės formulės ir jų taikymas
a — pirmasis narys b — antrasis narys
1

Sumos kvadratas

(a + b)2 = a2 + 2ab + b2
Taisyklė žodžiais Pirmojo nario kvadratas plius du kartus pirmasis kart antrasis plius antrojo nario kvadratas.
Pavyzdys
(x + 3)2 =
= x2 + 2·x·3 + 32 =
= x2 + 6x + 9
2

Skirtumo kvadratas

(a − b)2 = a2 − 2ab + b2
Taisyklė žodžiais Pirmojo nario kvadratas minus du kartus pirmasis kart antrasis plius antrojo nario kvadratas.
Pavyzdys
(2y − 5)2 =
= (2y)2 − 2·2y·5 + 52 =
= 4y2 − 20y + 25
3

Dviejų narių sumos ir skirtumo sandauga — kvadratų skirtumas

(a + b)(a − b) = a2 − b2
Taisyklė žodžiais Sumos ir skirtumo sandauga lygi pirmojo nario kvadratui minus antrojo nario kvadratas. Vidurinis narys išnyksta.
Pavyzdys
(x + 7)(x − 7) =
= x2 − 72 =
= x2 − 49
